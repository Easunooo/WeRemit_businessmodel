# WeChat Cross-border Remittance Business Value Dashboard

This project is a Vite + React dashboard for estimating the business value of WeChat cross-border remittance scenarios.

## Local Development

**Prerequisites:** Node.js 20+

1. Install dependencies:
   `npm install`
2. Start the dev server:
   `npm run dev`

## Build

Run a production build with:

`npm run build`

## Deployment

Pushing to `main` triggers the GitHub Actions workflow in `.github/workflows/deploy.yml`, which builds the site and deploys it to GitHub Pages.
